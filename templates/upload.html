How It Works

This tool effortlessly converts a Curriculum Vitae (CV) in PDF format into structured JSON data:

  1. Upload your CV in PDF format.
  2. Our tool processes the CV, extracting relevant details.
  3. View the processed JSON content for accuracy.
  4. Download the structured JSON data seamlessly.

Technologies Used

We harness cutting-edge technologies:

  • Flask: Serving user requests.
  • OpenAI's GPT-4: Precise CV content structuring.
  • PyPDF2: Text extraction from PDFs.

CV Formatting Guidelines

Your CV should:

  • Be clear, legible, and structured.
  • Feature sections like 'Education', 'Experience'.
  • Minimize complex graphical elements.
